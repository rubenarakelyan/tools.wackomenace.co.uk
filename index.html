This site is a faithful recreation of the old wackomenace tools site using web archives. Most of the pages and tools have been lost in the mists of time, but I have tried to preserve as much as possible. Things that are still available have been moved to GitHub, while everything else has a placeholder page that describes what was once there in case someone with an old link happens to visit. I haven't attempted to recreate any of the lost tools since none of them are particularly unique.

Tools

  • Teletext Services
    A downloadable list of Teletext pages in the UK.
  • vCardMaker
    Makes standard vCard electronic business cards using an easy-to-use form.
  • wackomenace MD5 convertor
    Calculates the MD5 hash of data (NON-REVERSIBLE ENCRYPTION).
    Please note that this script does not log usage.
  • wackomenace Country List
    Makes an XHTML list of all the countries of the world using the official ISO list.

Scripts

  • phpBB RSS Content Syndicator
    Turns the latest posts to a phpBB forum into an RSS feed.
  • Odeon UK Site Fixer
    Makes the Odeon cinema website work in Mozilla Firefox. Deprecated.
  • wackomenace Flickr Badge
    A truly flexible version of the Flickr badge.
  • wackomenace JavaScript Include
    Nicely includes several JavaScripts into your web pages.
  • wackomenace Random Colour Validator
    Generates and validates a random colour to stop spambots from submitting forms.

Hosted

  • Y i Believe Hosted Badges
    If you can't host your own Y i Believe badge, try this service.

Co-authored

  • Borland Pascal eXtensions
    Authored by William Setters and Ruben Arakelyan to extend the Pascal programming language with easy-to-use tools.

    A testing area for web pages and techniques. Stuff in here may be well and truly broken and is only recommended for experimentation!
